# codingTest_python

## 구현

+ 구현 유형의 문제

  풀이를 떠올리는 것은 쉽지만 소스코드로 옮기기 어려운 문제
  
+ 구현 유형의 문제 종류

  * 완전탐색
    
    모든 경우의 수를 주저없이 다 계산하는 해결방법
    
  * 시뮬레이션
  
    문제에서 제시한 알고리즘을 한 단계씩 차례대로 직접 수행
    
+ 파이썬에서 리스트 크기

  * int 자료형 데이터의 개수에 따른 메모리 사용량
   
    |데이터의 개수|메모리 사용량|
    |------|---|---|
    |1,000|약 4KB|
    |1,000,000|약 4MB|
    |10,000,000|약 40MB|
